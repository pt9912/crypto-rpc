# Offene Trigger-Watch-Punkte

Dieses Verzeichnis sammelt Trigger, offene Folgearbeiten und
Vorabklärungen, die noch keinen konkreten Scope haben.

Lebenszyklus und Verzeichnisstruktur sind in
[`ADR 0001`](../../adr/0001-documentation-and-planning-structure.md)
§2.4 definiert.

## Konvention für Einträge

- Dateiname `NNN-kurz-titel.md` mit dreistelliger Nummer.
- Jeder Eintrag beschreibt Trigger, Beobachtung und Aktivierungsbedingung
  (was muss passieren, damit der Eintrag nach `next/` wandert?).
- Einträge bleiben hier, bis sie aktiviert oder verworfen werden.
