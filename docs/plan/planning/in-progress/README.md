# Laufende Arbeit

Aktive Slice-Pläne und Roadmap-Dokumente. Jeder Eintrag muss
Akzeptanzkriterien und einen Verifikationspfad enthalten (siehe
[`ADR 0001`](../../adr/0001-documentation-and-planning-structure.md)
§4).

Lebenszyklus und Verzeichnisstruktur sind in
[`ADR 0001`](../../adr/0001-documentation-and-planning-structure.md)
§2.4 definiert.

## Konvention für Einträge

- Roadmap-Dokumente nutzen sprechende Namen (z. B. `roadmap.md`,
  `M1-mvp-pkcs11.md`).
- Slice-Pläne tragen `NNN-kurz-titel.md` (Nummer aus `next/` übernommen).
- Bei Abschluss wandert der Eintrag mit Closure-Notiz nach `done/`.
