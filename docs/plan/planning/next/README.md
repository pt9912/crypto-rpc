# Geplante Arbeit (Scope skizziert)

Einträge, deren Scope skizziert ist, die aber noch nicht aktiv
bearbeitet werden.

Lebenszyklus und Verzeichnisstruktur sind in
[`ADR 0001`](../../adr/0001-documentation-and-planning-structure.md)
§2.4 definiert.

## Konvention für Einträge

- Dateiname `NNN-kurz-titel.md` mit dreistelliger Nummer (ggf. aus
  `open/` übernommen).
- Jeder Eintrag enthält Ziel, grobes Scope, Akzeptanzkriterien und
  Vorbedingungen für die Aktivierung.
